Delegation guide (wallet-oriented)

Checklist je isti povsod: verify identity → delegate → confirm epoch rules.

1

Verify identity first

Open the Explorer/FlareMetrics links and confirm that the address matches the one on this page.

2

Paste the delegation address

Use “Copy delegation address” to avoid typing errors.

3

Confirm fees + epoch timing

The settings usually take effect in the next epoch. Do not expect instant changes.

4

Sanity-check after

Wait one epoch, refresh the metrics, and only then evaluate the result.

Wallet quick notes
  • MetaMask: double-check network + prompts before approving.
  • Bifrost: find delegation/FTSO section; verify provider identity via link.
  • Hardware wallets: confirm on-device display matches address/tx details.
  • Epoch timing: changes apply from next epoch; don’t panic if not instant.
  • Security: never share seed phrase; beware fake support DMs.

Extra checklist

  • Compare identity across Explorer + FlareMetrics.
  • Use copy buttons (avoid typos).
  • If something looks off, wait one epoch and re-check.
